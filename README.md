# Aura Mobile

This folder is the Expo/React Native migration scaffold for the current web frontend.

## Purpose
- Mirror the existing app features in a mobile-first Expo structure.
- Keep UI, navigation, state, API services, and realtime logic separated cleanly.
- Provide a safe starting point for converting the web codebase file by file.

## Current state
- The folder tree has been created.
- Implementation files are still placeholders and can be filled in next.

## Next step
- Create the Expo app entry files under `app/`.
- Add the shared `src/` modules for API, store, hooks, and components.
- Port features from the web frontend one area at a time.
